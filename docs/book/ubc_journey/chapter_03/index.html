    第三章：专业选择¶

    3.1 第一年结束¶

    第一年的学习很快结束了。五月份的时候，UBC的校园变得格外美丽，樱花盛开，绿草如茵。五个人都顺利完成了所有课程，但成绩各有不同。

    李明轩：GPA 3.8，表现优秀。他在所有课程中都取得了很好的成绩，特别是在化学和数学方面。

    陈雨婷：GPA 3.6，良好。她的数学成绩很好，但在生物和化学方面相对较弱。

    王浩：GPA 3.7，良好。他在物理和数学方面表现突出，但在生物方面稍显不足。

    张伟：GPA 3.4，中等。他的生物成绩很好，但在数学和物理方面需要提高。

    林晓梅：GPA 3.2，中等。她的成绩相对平均，没有特别突出的科目，也没有特别弱的科目。

    暑假期间，五个人都回到了中国。但他们没有完全放松，而是在思考自己的专业选择。UBC的第二年需要确定具体的专业，这是一个重要的决定，会影响到他们未来的职业发展。

    3.2 专业探索过程¶

    在暑假期间，五个人都做了大量的专业探索工作：

    李明轩的探索： 明轩利用暑假时间参加了北京的一个编程训练营。他发现自己对编程很有天赋，逻辑思维能力强，而且很享受解决问题的过程。他还和几个在IT行业工作的学长学姐交流，了解了计算机科学专业的就业前景。

    "计算机科学是一个很有前景的专业，"一个在Google工作的学长告诉他，"特别是在温哥华，有很多科技公司，就业机会很多。而且计算机科学的薪资水平也很高。"

    明轩还做了大量的在线研究，了解了UBC计算机科学专业的课程设置、入学要求、就业前景等等。他发现，UBC的计算机科学专业在加拿大很有名，毕业生很受雇主欢迎。

    陈雨婷的探索： 雨婷在上海参加了一个数学夏令营。在这里，她遇到了很多来自不同大学的数学爱好者。通过交流，她发现自己对应用数学更感兴趣，而不是纯数学。

    "应用数学更实用，"一个教授告诉她，"特别是与计算机科学结合的应用数学，就业前景很好。"

    雨婷还咨询了UBC的学术顾问，了解了不同专业的要求和前景。学术顾问建议她考虑计算机科学专业，因为她的数学背景很好，而且计算机科学对数学能力要求很高。

    王浩的探索： 王浩在深圳参加了一个科技创新营。在这里，他接触到了很多前沿的科技，比如人工智能、机器学习、区块链等等。他发现自己对这些技术很感兴趣，特别是人工智能。

    "AI是未来的趋势，"一个创业公司的CEO告诉他，"如果你对AI感兴趣，计算机科学专业是最好的选择。"

    王浩还参观了几家科技公司，了解了软件工程师的工作内容。他发现，这份工作不仅技术含量高，而且很有创造性，很符合他的兴趣。

    张伟的探索： 张伟在广州的一个医院做志愿者。在这里，他接触到了很多医学研究，发现生物化学在医学研究中应用很广。他还和几个医生交流，了解了生物化学专业的发展前景。

    "生物化学是一个很好的专业，"一个医生告诉他，"特别是在医学研究方面，有很多机会。如果你对研究感兴趣，这个专业很适合你。"

    张伟还了解到，UBC的生物化学专业很强，有很多优秀的研究机会。他开始考虑是否要走学术研究的道路。

    林晓梅的探索： 晓梅在南京的一个心理咨询中心做志愿者。在这里，她接触到了很多有心理困扰的人，发现心理咨询师的工作很有意义。她还和几个心理咨询师交流，了解了这个职业的要求和前景。

    "心理咨询是一个很有意义的职业，"一个资深心理咨询师告诉她，"你能够帮助很多人解决心理问题，看到他们的生活得到改善，这会给你带来很大的成就感。"

    晓梅还了解到，UBC的心理学专业很好，有很多实践机会。她开始认真考虑是否要选择心理学专业。

    3.3 专业选择决策过程¶

    回到温哥华后，五个人都做出了自己的专业选择。他们的决策过程包括以下几个步骤：

    自我评估： 每个人都做了详细的自我评估，包括自己的兴趣、能力、价值观、性格等等。他们使用了各种评估工具，比如MBTI性格测试、霍兰德职业兴趣测试等等。

    信息收集： 他们收集了大量关于不同专业的信息，包括课程设置、入学要求、就业前景、薪资水平等等。他们使用了各种信息来源，比如学校网站、学术顾问、学长学姐、行业报告等等。

    选项比较： 他们列出了几个可能的专业选项，然后从多个维度进行比较，比如兴趣匹配度、能力匹配度、就业前景、薪资水平、学习难度等等。

    决策制定： 基于自我评估、信息收集和选项比较，每个人都做出了自己的专业选择。他们还制定了备选方案，以防第一选择不能实现。

    3.4 专业选择结果¶

    经过深思熟虑，五个人都做出了自己的专业选择：

    李明轩 - 计算机科学 明轩在第一年的编程课（CPSC 110）中发现自己对编程很有天赋。他喜欢解决逻辑问题，而且计算机科学的前景很好。他的数学和物理成绩都很好，这为计算机科学的学习打下了良好的基础。

    "我选择计算机科学是因为我喜欢编程，喜欢解决问题，"明轩说，"而且计算机科学的就业前景很好，薪资水平也很高。"

    陈雨婷 - 计算机科学 雨婷原本想选数学，但在和学长学姐交流后，发现计算机科学更适合她。她喜欢编程的创造性和实用性，而且她的数学背景很好，这为计算机科学的学习提供了优势。

    "我选择计算机科学是因为它结合了数学和编程，"雨婷说，"而且计算机科学的应用很广泛，可以在很多行业工作。"

    王浩 - 计算机科学 王浩在编程俱乐部中深入了解了计算机科学，决定转入这个专业。他特别喜欢人工智能和机器学习，希望将来能够从事相关的工作。他的物理和数学成绩都很好，这为计算机科学的学习提供了基础。

    "我选择计算机科学是因为我对AI很感兴趣，"王浩说，"我希望将来能够开发出有意义的AI应用。"

    张伟 - 生物化学 张伟对生物和化学都很感兴趣，决定选择生物化学专业。他希望将来能够从事医学研究，为人类健康做出贡献。他的生物成绩很好，这为生物化学的学习提供了基础。

    "我选择生物化学是因为我对医学研究很感兴趣，"张伟说，"我希望将来能够开发出新的药物或治疗方法。"

    林晓梅 - 心理学 晓梅在第一年的探索中发现自己对人类行为很感兴趣。她选择了心理学专业，希望将来能够成为一名心理咨询师，帮助有心理困扰的人。她的沟通能力很好，这为心理学的学习和实践提供了优势。

    "我选择心理学是因为我喜欢帮助别人，"晓梅说，"我希望将来能够成为一名心理咨询师，帮助人们解决心理问题。"

    3.5 转专业挑战与准备¶

    对于选择计算机科学的三个人来说，转专业并不容易。UBC的计算机科学专业竞争很激烈，需要很高的GPA才能进入。根据学校的要求，转入计算机科学需要至少3.3的GPA，而且竞争很激烈。

    他们开始为转专业做准备：

    提高GPA： 第二年的课程更加困难，他们需要保持优秀的成绩。他们制定了详细的学习计划，每周安排足够的时间学习，参加学习小组，寻求tutor帮助等等。

    编程练习： 除了课程作业，他们还做了很多额外的编程练习。明轩参加了LeetCode的算法练习，每天做几道算法题。雨婷参加了HackerRank的编程挑战，提高自己的编程技能。王浩参加了Kaggle的数据科学竞赛，学习机器学习的应用。

    项目经验： 他们开始做一些个人项目，丰富自己的简历。明轩开发了一个简单的网站，展示自己的编程技能。雨婷开发了一个移动应用，帮助学生学习数学。王浩开发了一个简单的AI程序，识别图片中的物体。

    参加竞赛： 他们参加了ACM编程竞赛，虽然成绩一般，但学到了很多。通过竞赛，他们了解了团队合作的重要性，也提高了自己的编程能力。

    3.7 学术顾问的重要性¶

    在专业选择过程中，学术顾问起到了很重要的作用。UBC每个院系都有专门的学术顾问，帮助学生选择专业和规划学业。

    五个人都预约了学术顾问的咨询时间：

    李明轩的咨询： 明轩的学术顾问是计算机科学系的Dr. Chen。Dr. Chen详细了解了明轩的兴趣和能力，建议他选择计算机科学专业。他还告诉明轩关于转专业的具体要求和准备工作。

    "计算机科学专业竞争很激烈，"Dr. Chen说，"但你的背景很好，只要努力准备，很有希望成功转入。"

    陈雨婷的咨询： 雨婷的学术顾问是科学学院的Dr. Smith。Dr. Smith帮助雨婷分析了她的优势和劣势，建议她考虑计算机科学专业。他还告诉雨婷关于计算机科学专业的课程设置和就业前景。

    "你的数学背景很好，这为计算机科学的学习提供了优势，"Dr. Smith说，"而且计算机科学专业对女生的需求很大，就业前景很好。"

    王浩的咨询： 王浩的学术顾问是物理系的Dr. Johnson。Dr. Johnson了解到王浩对AI的兴趣，建议他选择计算机科学专业。他还告诉王浩关于AI领域的发展趋势和就业机会。

    "AI是未来的趋势，"Dr. Johnson说，"如果你对AI感兴趣，计算机科学专业是最好的选择。"

    张伟的咨询： 张伟的学术顾问是生物系的Dr. Brown。Dr. Brown了解到张伟对医学研究的兴趣，建议他选择生物化学专业。他还告诉张伟关于生物化学专业的研究机会和就业前景。

    "生物化学专业有很多研究机会，"Dr. Brown说，"如果你对研究感兴趣，这个专业很适合你。"

    林晓梅的咨询： 晓梅的学术顾问是心理学系的Dr. Lee。Dr. Lee了解到晓梅对帮助他人的兴趣，建议她选择心理学专业。她还告诉晓梅关于心理学专业的实践机会和职业发展。

    "心理学专业有很多实践机会，"Dr. Lee说，"如果你喜欢帮助别人，这个专业很适合你。"

    3.8 家长意见与沟通¶

    专业选择不仅是学生自己的决定，也需要考虑家长的意见。五个人都和父母进行了深入的沟通：

    李明轩的家庭沟通： 明轩的父母都是大学教授，很支持他的选择。他们认为计算机科学是一个很好的专业，就业前景很好。他们还表示会继续支持明轩的学业。

    "我们相信你的判断，"明轩的父亲说，"计算机科学是一个很好的专业，我们会全力支持你。"

    陈雨婷的家庭沟通： 雨婷的父母有些担心，因为他们希望女儿选择一个更稳定的专业。但在雨婷详细解释了计算机科学的前景后，他们也表示支持。

    "只要你喜欢，我们就支持，"雨婷的母亲说，"但你要努力学习，不要让我们失望。"

    王浩的家庭沟通： 王浩的父母都是工程师，很支持他选择计算机科学。他们认为这个专业很适合王浩，而且就业前景很好。

    "我们为你感到骄傲，"王浩的父亲说，"计算机科学是一个很好的专业，我们会支持你。"

    张伟的家庭沟通： 张伟的父母对生物化学专业不太了解，有些担心就业前景。但在张伟详细解释后，他们也表示支持。

    "我们相信你的选择，"张伟的母亲说，"只要你努力学习，我们就支持你。"

    林晓梅的家庭沟通： 晓梅的父母都是教师，很支持她选择心理学。他们认为这个专业很有意义，而且很适合晓梅的性格。

    "我们为你感到高兴，"晓梅的父亲说，"心理学是一个很有意义的专业，我们会支持你。"

    3.9 专业选择的时间规划¶

    为了确保能够成功转入理想的专业，五个人都制定了详细的时间规划：

    第二年的规划： - 第一学期：保持优秀的GPA，参加编程练习，做个人项目 - 第二学期：继续提高GPA，参加编程竞赛，寻求实习机会 - 暑假：参加实习，积累工作经验，准备转专业申请

    转专业申请的准备： - 准备个人陈述，说明选择计算机科学的原因 - 准备简历，展示自己的编程技能和项目经验 - 联系教授，寻求推荐信 - 准备面试，练习常见的面试问题

    3.10 专业选择的风险管理¶

    五个人都意识到专业选择有一定的风险，他们制定了风险管理策略：

    备选方案： 每个人都准备了备选方案，以防第一选择不能实现。明轩的备选方案是数学专业，雨婷的备选方案是统计学专业，王浩的备选方案是物理专业。

    技能培养： 他们注重培养可转移的技能，比如编程能力、分析能力、沟通能力等等。这些技能在任何专业都很有用。

    灵活调整： 他们保持开放的心态，愿意根据情况调整自己的计划。如果发现选择的专业不适合自己，他们会考虑转专业。

    3.11 本章小结¶

    第三章讲述了五个学生在UBC的专业选择过程。通过他们的经历，我们可以学到：

    1. 专业选择的重要性：专业选择会影响到未来的职业发展，要认真对待。

    2. 自我评估的必要性：要了解自己的兴趣、能力、价值观、性格等等，选择适合自己的专业。

    3. 信息收集的重要性：要收集充分的信息，了解不同专业的要求和前景。

    4. 学术顾问的价值：学术顾问能够提供专业的建议和指导，要充分利用这个资源。

    5. 家长沟通的重要性：专业选择不仅是学生自己的决定，也需要考虑家长的意见。

    6. 风险管理的必要性：要制定备选方案，培养可转移的技能，保持灵活调整的能力。

    在下一章中，我们将看到这五个学生如何在各自的专业中学习和成长，如何面对各种挑战，如何为自己的未来做准备。
